CPH Cloud - Competitive Programming Helper

⚠️ WARNING: If you have the original CPH extension installed and enabled, please disable or uninstall it and restart VSCode before installing and using this extension!

CPH Cloud is a Visual Studio Code extension that allows you to quickly compile, run and judge competitive programming problems from within the IDE using cloud-based compilation. You can automatically download testcases or write & test your own problems. No local compiler setup required!

About: This is a fork of the original CPH extension by Divyanshu Agrawal. The frontend interface is based on the original, but the entire backend (cloud API integration, compilation logic, execution handling) is original work.


Features

  • Cloud-based compilation - No local compiler setup required
  • Platform-independent - Works on any OS
  • Automatic compilation with display for compilation errors.
  • Intelligent judge with support for signals, timeouts and runtime errors.
  • Works with Competitive Companion.
  • Quick Run button in editor title bar for fast testing
  • Support for several languages: C, C++, Python, Java, JavaScript, Rust, Go, C#
